1. USUARIO A (Referente):
• Genera enlace: https://t.me/bot?start=ref_123456
• Comparte enlace con amigos

2. USUARIO B (Referido):
• Usa enlace de referido
• Se registra como PENDIENTE
• DEBE conectar cuenta + hacer 1 descarga

3. CONFIRMACIÓN AUTOMÁTICA:
• Cuando USUARIO B cumple requisitos
• Sistema confirma referido
• USUARIO A recibe +1 en contador

4. RECOMPENSA:
• Cada 15 referidos confirmados = 1 día Premium
• Máximo 15 días acumulables
• Se aplica automáticamente
